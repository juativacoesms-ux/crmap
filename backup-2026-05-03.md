## Regenerar arte embutida (se trocar o modelo PNG)

Ficheiro local `carteirinha-base.png` (não versionado). Comando:

```bash
node -e "const fs=require('fs'); const b=fs.readFileSync('carteirinha-base.png'); fs.writeFileSync('carteirinha-base-data.js','window.__CRMAP_CARTEIRINHA_BASE_DATA_URL__='+JSON.stringify('data:image/png;base64,'+b.toString('base64'))+';');"
```
